1.Объект - карандаш.
  Состояние :
    острота кончика грифеля.
  Поведение :
    рисовать (при использовании затупляется кончик грифеля)

2.Объект - выключатель света.
   Состояние :
    вкл. выкл.
   Поведение :
    включить - При включении начинает гореть люстра.
    выключить - При выключении люстра гаснет.

3.Объект - графин с водой.
   Состояние :
    количество воды.
   Поведение :
    налив воды в стакан - При использовании уменьшается количество воды.
    
4.Объект - обогреватель.
  Состояние :
   вкл. выкл.
   температура корпуса.
  Поведение :
   включить - При включении начинает повышаться температура корпуса.
   выключить - При выключении температура корпуса снижается.

5.Объект - ластик.
   Состояние :
    длина ластика.
   Поведение :
    стирание карандаша - При использовании уменьшается длиналастика
    
 6.Объект - ксерокс.
  состояние :
    вкл. выкл.
    количество тонера
    количество бумаги в лотке
  Поведение :
    включить - При включении происходит копирование документа, уменьшение количества тонера и объема бумаги.
    
 7.Объект - светофор
  состояние :
    зеленый свет - вкл./выкл./мигающий
    желтый свет - вкл./выкл./мигающий
    красный свет - вкл./выкл.
    движение автомобиля
   Поведение :
    зелевый свет (вкл.), желтый (выкл.), красный (выкл.) - автомобиль движется;
    зелевый свет (миг.), желтый (выкл.), красный (выкл.) - автомобиль движется или останавливается;
    зелевый свет (выкл.), желтый (вкл.), красный (выкл.) - автомобиль останавливается или стоит;
    зелевый свет (выкл.), желтый (выкл.), красный (вкл.) - автомобиль стоит;
    зелевый свет (выкл.), желтый (вкл.), красный (вкл.) - автомобиль стоит;
    зелевый свет (выкл.), желтый (миг.), красный (выкл.) - автомобиль движется в соответствии с приоритетом.
    
  8.Объект - кнопка дверного звонка
    состояние :
      вкл./выкл.
      сигнал звонка
    Поведение :
      кнопка включена - Подача сигнала звонка (световали и/или звуковая)
      кнопка выключена - ничего не происходит
      
  9.Объект - эскалатор
    состояние :
      вкл./выкл.
      пешеход идет/стоит
     Поведение:
      Экалатор вкл. - Происходит движение эскалатора при этом пешеход стоит
      Эскалатор выкл. - Эскалатор не движется при этом пешеход поднимается/спускается самостоятельно
      
   10.объект - компрессор
      состояние :
        вкл./выкл.
        состояние автомобильной шины
      Поведение :
        компрессор включен - При работе компрессора происходит накачка автомобильной шины
        компрессор выключен - ничего не происходит
